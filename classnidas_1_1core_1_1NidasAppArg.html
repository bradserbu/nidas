Detailed Description

A NidasAppArg identifies a particular kind of command-line argument which can be handled by NidasApp.

The base class defines basic state and behavior about the argument, then each type of argument has its own instance member in the NidasApp. Arguments can be subclassed from this class to provide extra customization. Only NidasApp can create instances of NidasAppArg.
