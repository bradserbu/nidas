Detailed Description

The Logging facility can store multiple logging configuration schemes, each with a particular name.

Each scheme is a list of LogConfig objects. These methods are used to manipulate the set of schemes known by the Logger. At any time the logging configuration can switch schemes by calling setScheme(). The first time a new scheme name is referenced, a default LogScheme is created with that name.
